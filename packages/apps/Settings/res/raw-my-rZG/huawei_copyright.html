အေရးႀကီးသည့္ မွတ္ခ်က္ : ဤသေဘာတူညီခ်က္သည္ သင္ (တသီးပုဂၢလ၊ ကုမၸဏီ၊ သို႔မဟုတ္ အျခား အဖြဲ႔အစည္း တစ္ခုခု) ႏွင့္ Huawei Device Co., Ltd. (ဤေနရာတြင္ "Huawei" အျဖစ္ ညႊန္းဆိုထားသည္) တုိ႔အၾကား ခ်ဳပ္ဆုိထားသည့္ ေနာက္ဆံုးသံုးစြဲသူ ေဆာ့ဖ္ဝဲလုိင္စင္ သေဘာတူညီခ်က္ (ဤေနရာတြင္ "သေဘာတူညီခ်က္" အျဖစ္ ညႊန္းထားသည္) ျဖစ္သည္။ (ဤသေဘာတူခ်က္၏ အပိုဒ္ 1.1 ကိုၾကည့္႐ႈပါ)။ သင္၏ Huawei ကိရိယာကို အသံုးမျပဳမီ သို႔မဟုတ္ ဤသေဘာတူညီခ်က္တြင္ ေဖာ္ျပထားသည့္ ေဆာ့ဖ္ဝဲ အသစ္မြမ္းမံမႈတစ္ခုကို ေဒါင္းလုဒ္မဆြဲမီ ယခု သေဘာတူညီခ်က္ကို ဂ႐ုတစိုက္ျဖင့္ ဖတ္႐ႈပါ။ Huawei ကိရိယာ သို႔မဟုတ္ ဤေဆာ့ဖ္ဝဲ အသစ္မြမ္းမံမႈ ေဒါင္းလုဒ္ကို အသံုးျပဳျခင္းကို ဤသေဘာတူညီခ်က္တြင္ ေဖာ္ျပထားသည့္ ပါဝင္ခ်က္မ်ားအား သင့္အေနျဖင့္ လက္ခံထားသည္အျဖစ္ မွတ္ယူမည္ျဖစ္သည္။
ထိုသို႔မဟုတ္ပါက Huawei ကိရိယာတစ္ခုခုကို အသံုးမျပဳပါႏွင့္ သို႔မဟုတ္ ဤေဆာ့ဖ္ဝဲ အသစ္မြမ္းမံမႈကို ေဒါင္းလုဒ္မဆြဲပါႏွင့္။
ဤသေဘာတူညီခ်က္ကို လက္မခံလွ်င္ အပိုဒ္ 6 တြင္ေဖာ္ျပထားသည့္ ေဒတာသံုးစြဲမႈ သေဘာတူညီခ်က္အပါအဝင္ Huawei ၏ ပစၥည္းျပန္ပို႔ေရး မူဝါဒႏွင့္ ကိုက္ညီေနသမွ် ေငြျပန္ေပးရန္အတြက္ သင္၏ Huawei ကိရိယာကို အခြင့္ရ စတိုးဆိုင္ (သို႔) ဝယ္ယူခဲ့ေသာ ခြင့္ျပဳခ်က္ရရွိထားသည့္ ျဖန္႔ျဖဴးသူထံသို႔ ျပန္လည္အပ္ႏွံႏိုင္သည္။
၎မူဝါဒ၏ စည္းမ်ဥ္းမ်ားသည္ ဝယ္ယူသည့္ ႏိုင္ငံ/ေဒသပိုင္း၏ သက္ဆိုင္ရာ ဥပေဒမ်ားႏွင့္ ထိန္းခ်ဳပ္ထားခ်က္တို႔ႏွင့္အညီျဖစ္သည္။
ဤသေဘာတူညီခ်က္တြင္ ေဖာ္ျပထားသည့္ ေဆာ့ဖ္ဝဲဆိုသည္မွ အပိုဒ္ 1.2 တြင္ ဖြင့္ဆုိေဖာ္ျပထားသည့္ Open Source ေဆာ့ဖ္ဝဲ အစိတ္အပိုင္းမ်ား မပါဝင္ပါ။ သင္သည္ ဤသေဘာတူညီခ်က္၏ မိတၱဴကို ဟြာေဝး၏ ဝက္ဘ္ဆုိက္ (consumer.huawei.com) မွမည္သည့္အခ်ိန္တြင္မဆို ဝင္ေရာက္ဖတ္႐ႈႏုိင္ၿပီး ေနာက္ပိုင္းရည္ၫႊန္းရန္အတြက္ ဤသေဘာတူညီခ်က္၏ မိတၱဴကို ရယူထားႏုိင္သည္။

1. အေထြေထြ စည္းမ်ဥ္းမ်ား
1.1 ဤသေဘာတူညီခ်က္ေအာက္မွ ေဆာ့ဖ္ဝဲအေနျဖင့္ ရည္ညႊန္းသည္မွာ : (1) ျမႇဳပ္သြင္းထားသည့္ ေဆာ့ဖ္ဝဲ၊ မွတ္တမ္းမ်ား၊ ၾကားခံစနစ္မ်ား၊ ပါဝင္ခ်က္မ်ား၊ ေဖာင့္မ်ားႏွင့္ Huawei သို႔မဟုတ္ ၎၏ လိုင္စင္ထုတ္ေပးသူမွ ပိုင္ဆိုင္သည့္ မူပိုင္ခြင့္ဥပေဒျဖင့္ ကာကြယ္ထားသည့္ ယင္းတြင္ ထည့္သြင္းထားသည့္ ေဒတာတစ္ခုခုတို႔ကဲ့သို႔ ကုဒ္မ်ားအထိ ကန္႔သတ္ခ်က္မရွိတို႔အပါအဝင္ Huawei ကိရိယာတစ္ခုႏွင့္အတူ သင့္ထံသို႔ ပံ့ပိုးလိုက္သည့္ ေဆာ့ဖ္ဝဲ ပါဝင္ပစၥည္းမ်ား၊ (2) (1)တြင္ ေဖာ္ျပထားသည့္တစ္ခုခုအတြက္ အသစ္မြမ္းမံမႈမ်ား သို႔မဟုတ္ အဆင့္ျမႇင့္တင္မႈမ်ားတို႔ျဖစ္သည္။
ေဆာ့ဖ္ဝဲတြင္ အပိုဒ္ 1.2 တြင္ေဖာ္ျပထားသည့္ Open Source အစိတ္အပိုင္းမ်ား မပါဝင္ပါ။
1.2 Huawei ကိရိယာ သို႔မဟုတ္ ၎၏ အသစ္မြမ္းမံမႈမ်ား သို႔မဟုတ္ အဆင့္ျမႇင့္တင္မႈမ်ားတြင္ Open Source ေဆာ့ဖ္ဝဲ လိုင္စင္ခြင့္ျပဳခ်က္မ်ား ("Open Source ေဆာ့ဖ္ဝဲ အစိတ္အပိုင္းမ်ား") ေအာက္မွ အဆိုပါ ေဆာ့ဖ္ဝဲ အစိတ္အပိုင္းမ်ားကို လုပ္ပိုင္ခြင့္ျပဳထားသည့္ ၾကားခံအဖြဲ႔မ်ားမွ ထားရွိေသာ မူပိုင္ခြင့္ဥပေဒမ်ားျဖင့္ကာကြယ္ထားသည့္ ေဆာ့ဖ္ဝဲ ပါဝင္ပစၥည္းမ်ားလည္း အပါအဝင္ျဖစ္ႏိုင္သည္။
စက္ပစၥည္းအတြင္းတြင္ ပံ့ပိုးထားသည့္ သီးျခားလိုင္စင္အခ်က္အလက္မွတဆင့္ကဲ့သို႔ အဆိုပါ Open Source ေဆာ့ဖ္ဝဲ အစိတ္အပိုင္းမ်ားကို သက္ဆုိင္ရာ ပါဝင္ပစၥည္းအလိုက္ Open Source ေဆာ့ဖ္ဝဲလုိင္စင္တုိ႔အား က်င့္သံုးႏုိင္သည္။
ထုိအေၾကာင္းေၾကာင့္ Open Source လုိင္စင္မ်ား၏ စည္းကမ္းမ်ားႏွင့္ သေဘာတူညီခ်က္မ်ားသည္ အဆုိပါ Open Source ေဆာ့ဖ္ဝဲ အစိတ္အပိုင္းမ်ားကို သင္အသံုးျပဳထားစဥ္ သက္ေရာက္မႈ႐ွိသင့္သည္။
ဤသေဘာတူညီခ်က္သည္ ယင္းကဲ့သို႔ Open Source ေဆာ့ဖ္ဝဲ လိုင္စင္ခြင့္ျပဳခ်က္မ်ားေအာက္တြင္ ရွိႏိုင္သည့္ ရပိုင္ခြင့္မ်ားႏွင့္ ဝတၱရားမ်ားကို ေျပာင္းလဲမႈမရွိပါ။
ဤသေဘာတူညီခ်က္တြင္ ေဖာ္ျပထားသည့္ လိုင္စင္ သတ္မွတ္ခ်က္မ်ားႏွင့္ Open Source ေဆာ့ဖ္ဝဲ အစိတ္အပိုင္းတစ္ခုခုႏွင့္ သက္ဆိုင္သည့္ Open Source ေဆာ့ဖ္ဝဲ လိုင္စင္မ်ား၏ ပါဝင္ခ်က္မ်ားတို႔အၾကား ပဋိပကၡတစ္ခုခုျဖစ္ပြားလွ်င္ Open Source ေဆာ့ဖ္ဝဲ လိုင္စင္မ်ားက လႊမ္းမိုးသြားမည္ျဖစ္သည္။
1.3 သေဘာတူညီမႈ မရွိသည့္တိုင္ ဤသေဘာတူညီခ်က္ေအာက္မွ ေဆာ့ဖ္ဝဲကို ဖတ္႐ႈရန္သာျဖစ္သည့္ မွတ္ဥာဏ္တစ္ခု သို႔မဟုတ္ အျခား မီဒီယာ သို႔မဟုတ္ ပံုစံတစ္ခုခုအတြင္း သိမ္းဆည္းထားသည္ ျဖစ္ေစ၊ သို႔မဟုတ္ Huawei မွ ခြင့္ျပဳထားသည့္ အြန္လိုင္း ေဒါင္းလုဒ္ တည္ေနရာတစ္ခုမွ ရယူထားသည္ျဖစ္ေစ ၎မွ ကန္႔သတ္ကာကြယ္ထားသည္။

2. ေနာက္ဆံုး အသံုးျပဳသူ ေဆာ့ဖ္ဝဲ လိုင္စင္
ဤေဆာ့ဖ္ဝဲကို (အပိုဒ္ 1.1 တြင္ ဖြင့္ဆိုထားသည့္အတိုင္း) ဤသေဘာတူညီခ်က္၏ စည္းမ်ဥ္းစည္းကမ္းႏွင့္ သတ္မွတ္ခ်က္မ်ားကို လိုက္နာသည့္ အေျခအေနျဖင့္ ကန္႔သတ္ထားသည့္ ဘက္စံုမဟုတ္ေသာ နည္းလမ္းျဖင့္ Huawei ကိရိယာ သို႔မဟုတ္ ကြန္ပ်ဴတာ တစ္လံုးနည္းေပၚတြင္အသံုးျပဳရန္ သင့္ကို ခြင့္ျပဳထားသည္။
၎ကို ေရာင္းခ်ျခင္း၊ လႊဲေျပာင္းျခင္း သို႔မဟုတ္ အျခားေသာ ေရာင္းဝယ္မႈဆိုင္ရာ ရည္ရြယ္ခ်က္ တစ္ခုခုအတြက္ အသံုးျပဳျခင္း မလုပ္ႏိုင္ပါ။

3. အသံုးျပဳမႈ ကန္႔သတ္ျခင္း
3.1 သင့္အေနျဖင့္ Huawei နည္းပညာမ်ား သို႔မဟုတ္ Huawei ဉာဏဆုိင္ရာ ပိုင္ဆိုင္မႈမ်ားကို ဒီဇိုင္းပံုေဖာ္ရန္၊ ဖြံ႔ၿဖိဳးလုပ္ေဆာင္ရန္၊ ထုတ္လုပ္ရန္၊ ေရာင္းခ်ရန္ သို႔မဟုတ္ Huawei ကိရိယာမ်ားႏွင့္/သို႔မဟုတ္ ေဆာ့ဝဲမ်ားႏွင့္ ဆက္ႏြယ္သည့္ ၾကားခံအဖြဲ႔ ကိရိယာမ်ား/တြဲသံုးပစၥည္းမ်ား ခြင့္ျပဳရန္၊ အက္ပလီေကးရွင္း ေဆာ့ဖ္ဝဲ ဖြံ႔ၿဖိဳးလုပ္ေဆာင္ရန္တို႔အတြက္ ဟြာေဝး၏သေဘာတူညီခ်က္မပါဘဲ သေဘာတူညီမႈမရွိသည့္တိုင္ အသံုးျပဳ၍ မရႏိုင္ပါ။
3.2 မွတ္တမ္းသိမ္းျခင္းရည္ရြယ္ခ်က္အတြက္ ကြန္ပ်ဴတာ၏ ဟာ့ဒစ္စ္ သို႔မဟုတ္ အျခား သိုေလွာင္ခန္း မီဒီယာပစၥည္းတစ္ခုေပၚတြင္ ေဆာ့ဖ္ဝဲ မိတၱဴတစ္ခု ျပဳလုပ္ႏိုင္ပါသည္။
ေဆာ့ဖ္ဝဲသည္ ကိရိယာမ်ား သို႔မဟုတ္ ကြန္ပ်ဴတာမ်ားစြာတို႔အတြက္ တစ္ၿပိဳင္တည္း ရရွိေစႏိုင္သည့္အတြက္ ၎ကို ကြန္ယက္မ်ားတြင္ ျဖန္႔ျဖဴးျခင္း သို႔မဟုတ္ ေဖာ္ျပျခင္း မျပဳရပါ။

ဆုိလိုသည္က HUAWEI ၏ ထုတ္ကုန္ သို႔မဟုတ္ ဝန္ေဆာင္မႈမ်ားအား သင္အသံုးျပဳသည့္ ႏုိင္ငံ၏ျပင္ပ႐ွိ အျခားတရားစီရင္ႏုိင္ေသာ အဖြဲ႔အစည္းမ်ားသို႔ ေဒတာအား လႊဲေျပာင္းျခင္း၊ သို႔မဟုတ္ အဆိုပါအဖြဲ႔မ်ားက ဝင္ေရာက္အသံုးျပဳျခင္းမ်ားကို ျပဳလုပ္ႏိုင္သည္။
အဆုိပါတရားစီရင္ႏုိင္သည့္ အဖြဲ႔အစည္းမ်ားသည္ မတူညီသည့္ ေဒတာကာကြယ္မႈဥပေဒမ်ား႐ွိေနႏုိင္သည္၊ သို႔မဟုတ္ အဆိုပါဥပေဒမ်ား လံုးဝမ႐ွိျခင္းလည္းျဖစ္ႏုိင္သည္။
အဆုိပါအေျခအေနမ်ားတြင္ HUAWEI သည္ က်င့္သံုးေနဆဲ ဥပေဒမ်ားႏွင့္ စည္းမ်ဥ္းစည္းကမ္းမ်ားအရ လိုအပ္သည့္အတုိင္း ေဒတာအား ကာကြယ္မႈေပးႏုိင္သည့္ အလားတူ တူညီေသာ သို႔မဟုတ္ သင့္ေတာ္သည့္အဆင့္တစ္ခု႐ွိသည့္ ဥပေဒ႐ွိေနေၾကာင္း ေသခ်ာစစ္ေဆးမည္။
6.4  HUAWEI သည္သင့္ေဒတာကို ဤသေဘာတူညီခ်က္ႏွင့္ ပုဂၢိဳလ္ေရးရာထိန္းသိမ္းမႈမူဝါဒတြင္ ေဖာ္ျပထားသည့္အတုိင္း ရည္႐ြယ္ခ်က္တစ္ခုကို ၿပီးစီးသည့္အခ်ိန္အထိ ထိန္းသိမ္းထားမည္ျဖစ္ၿပီး ဥပေဒကခြင့္ျပဳထားသည္မွအပ လုိအပ္သည့္အခ်ိန္ကာလတစ္ခုထက္ ပို၍ထိန္းသိမ္းထားျခင္းမ်ားမျပဳလုပ္ပါ။
သင့္ေဒတာကိုေပါက္ၾကားေစႏုိင္သည့္ ခြင့္ျပဳခ်က္မ႐ွိေသာ ဝင္ေရာက္အသံုးျပဳမႈမ်ားကို ကာကြယ္ရန္ HUAWEI သည္သင့္ေတာ္ေသာ နည္းပညာႏွင့္ လုပ္ငန္းပိုင္းဆုိင္ရာအဆင့္မ်ားကို ရယူမည္ျဖစ္သည္၊ ေက်းဇူးျပဳ၍ တစ္ခုသိထားရမည္မွာ ၿပီးျပည့္စံုသည့္ လံုၿခံဳေရးစနစ္ဆိုသည္မွာ မ႐ွိေၾကာင္းျဖစ္သည္။
6.5  သေဘာတူညီခ်က္၏ ဤေၾကျငာခ်က္ကို လ်င္ျမန္ေသာသက္ေရာက္မႈျဖင့္ မည္သည့္အခ်ိန္တြင္မဆုိ ႐ုတ္သိမ္းႏုိင္သည္။ ႐ုတ္သိမ္းလုိက္ပါက အပိုဒ္ 7 တြင္ေဖာ္ျပထားသည့္ ဤသေဘာတူညီခ်က္အား အဆံုးသတ္ျခင္းက သက္ေရာက္မႈ႐ွိမည္။

7. အဆံုးသတ္ျခင္းႏွင့္ ဆက္လက္ အက်ဳံးဝင္သက္ေရာက္မႈ
7.1 ဤသေဘာတူညီခ်က္သည္ ေဆာ့ဖ္ဝဲကို ထည့္သြင္းသည့္ ပထမဆံုး ေန႔ရက္မွစ၍ အက်ဳံးဝင္ပါသည္။
Huawei မွပံ့ပိုးထားသည့္ ေဆာ့ဖ္ဝဲ၊ အရံသိမ္း ကူးယူထားမႈအားလံုးႏွင့္ သက္ဆိုင္ရာ ပစၥည္းအလံုးစံုတို႔ကို လံုးဝ ဖ်က္ပစ္ျခင္း၊ ဖ်က္ဆီးပစ္ျခင္းႏွင့္ သင့္ကုန္က်စရိတ္ျဖင့္ ျပန္လည္အပ္ႏွံမႈ ျပဳျခင္းျဖင့္ ဤသေဘာတူညီခ်က္ကို မည္သည့္အခ်ိန္တြင္မဆို အဆံုးသတ္ႏိုင္ပါသည္။
သင့္အေနျဖင့္ ဤသေဘာတူညီခ်က္၏ စည္းမ်ဥ္းစည္းကမ္းႏွင့္ သတ္မွတ္ခ်က္ တစ္ခုခုတို႔ႏွင့္အညီ လုပ္ေဆာင္ရန္ ပ်က္ကြက္ခဲ့ပါက Huawei သို႔မဟုတ္ ၎၏ လိုင္စင္ထုတ္ေပးသူမ်ားသည္ ဤသေဘာတူညီခ်က္ကို ႀကိဳတင္ေၾကညာမႈမရွိဘဲ မည္သည့္အခ်ိန္တြင္မဆို ရပ္စဲႏိုင္ပါသည္။
ရပ္ဆိုင္းၿပီးသည့္အခါ ဤေဆာ့ဖ္ဝဲ အသံုးျပဳမႈကို ရပ္စဲၿပီး သင္၏ Huawei ကိရိယာ သို႔မဟုတ္ ကြန္ပ်ဴတာတြင္ ကူးယူထားသည့္ ႏွင့္/သို႔မဟုတ္ ထည့္သြင္းထားသည့္ ေဆာ့ဖ္ဝဲအားလံုးႏွင့္ သက္ဆိုင္ရာ ပစၥည္းတို႔ကို ခ်က္ခ်င္း ဖ်က္ပစ္ရမည္ျဖစ္သည္။
7.2  ဤသေဘာတူညီခ်က္ ရပ္စဲၿပီးသည့္ေနာက္ အပိုဒ္ 6၊ 8၊ 9၊ 10၊ 11၊ 13 ႏွင့္ 14 တို႔ ဆက္လက္ အက်ဳံးဝင္ေနပါသည္။

8. အာမခံမ်ား ျငင္းဆိုခ်က္
8.1 သင့္ေလ်ာ္ကိုက္ညီမႈ အာမခံခ်က္မ်ားတို႔အပါအဝင္ မည္သည့္ပံုစံျဖင့္မဆို ေဖာ္ျပ သို႔မဟုတ္ သက္ေရာက္ေစသည့္ အာမခံခ်က္ တစ္စံုတစ္ရာမရွိဘဲ မည္သည့္ သက္ဆိုင္ရာ ဥပေဒကမွ် အျမင့္ဆံုး တိုးခ်ဲ႔ ခြင့္ျပဳထားသည္တို႔မရွိဘဲ "ထိုသို႔ေဖာ္ျပထားသည့္အတုိင္း" ပံ့ပိုးထားေၾကာင္း သင့္အေနျဖင့္ အသိအမွတ္ျပဳသေဘာတူရမည္။ HUAWEI ျဖစ္ေစ၊ ၎၏ လိုင္စင္ထုတ္ေပးသူမ်ား သို႔မဟုတ္ မိတ္ဘက္ ကုမၸဏီမ်ားျဖစ္ေစ၊ မည္သည့္ကိုယ္စားျပဳမႈကိုမဆို မူပိုင္ခြင့္ကိုင္ထားသူမ်ားျဖစ္ေစ သို႔မဟုတ္ အာမခံခ်က္မ်ား၊ ေဖာ္ျပထားခ်က္မ်ား၊ သက္ဆုိင္မႈမ်ားအပါအဝင္ ၄င္းတို႔၏ ေဆာ့ဖ္ဝဲကို သီးျခား ရည္ရြယ္ခ်ယ္၊ တိက်မႈ၊ ႏွစ္သက္မႈမရွိျခင္းႏွင့္ ၾကားခံအဖြဲ႔ ရပိုင္ခြင့္မ်ား၏ ထိပါးခ်ဳိးေဖာက္မႈတို႔အတြက္ ကူးသန္းေရာင္းဝယ္ျခင္း၊ အရည္အေသြး ေက်နပ္မႈရွိျခင္းတုိ႔အထိ အကန္႔အသတ္မ႐ွိေစရ။
ေဆာ့ဖ္ဝဲအတြင္း ပါရွိသည့္ လုပ္ေဆာင္ခ်က္မ်ားသည္ သင့္လိုအပ္ခ်က္မ်ားကို ျပည့္မီလိမ့္မည္ သို႔မဟုတ္ ေဆာ့ဖ္ဝဲ၏ လုပ္ေဆာင္ခ်က္သည္ အေႏွာင့္အယွက္မရွိ သို႔မဟုတ္ အမွားကင္းမည္ဟု HUAWEI သို႔မဟုတ္ အျခား အဖြဲ႔တစ္ခုခုမွ အာမခံခ်က္မရွိပါ။
သင္၏ ရည္မွန္းထားသည့္ ရလဒ္မ်ားေဖာ္ေဆာင္ရန္ ေဆာ့ဖ္ဝဲကို ေရြးခ်ယ္မႈအတြက္ႏွင့္ ထည့္သြင္းျခင္း၊ အသံုးျပဳျခင္းႏွင့္ ၎မွရရွိခဲ့သည့္ ရလဒ္မ်ားအတြက္ တာဝန္ရွိမႈႏွင့္ အလားအလာအားလံုးကို သင္လက္ခံရမည္။
8.2 ဤေဆာ့ဖ္ဝဲကို ထည့္သြင္းျခင္းသည္ ၾကားခံအဖြဲ႔ ေဆာ့ဖ္ဝဲ၊ အက္ပလီေကးရွင္း ပ႐ိုဂရမ္မ်ား သို႔မဟုတ္ ဝန္ေဆာင္မႈမ်ား၏ ရရွိႏိုင္မႈကို သက္ေရာက္ႏိုင္ပါသည္။
ဤေဆာ့ဖ္ဝဲတြင္ လုပ္ေဆာင္ခ်က္မ်ား သို႔မဟုတ္ ဆက္လက္၍ အစဥ္ျဖစ္သည့္ ဝန္ေဆာင္မႈမ်ား ရွိလိမ့္မည္ဟူသည္ ဝန္ေဆာင္မႈမ်ား သို႔မဟုတ္ ဤေဆာ့ဖ္ဝဲသည္ ၾကားခံအဖြဲ႔ ေဆာ့ဖ္ဝဲ သို႔မဟုတ္ ဝန္ေဆာင္မႈမ်ားႏွင့္ ကိုက္ညီလိမ့္မည္ဟု HUAWEI မွ အာမခံခ်က္မရွိပါ။
ထို႔အျပင္ ဤေဆာ့ဖ္ဝဲသည္ အျခားမည္သည့္ ၾကားခံ ေဆာ့ဖ္ဝဲ သို႔မဟုတ္ ဝန္ေဆာင္မႈမ်ားျဖင့္ ကိုက္ညီမႈ႐ွိမည္ျဖစ္ေၾကာင္း အာမ မခံပါ။
8.3 HUAWEI ေဆာ့ဖ္ဝဲႏွင့္ ဝန္ေဆာင္မႈမ်ားသည္ မွားယြင္းမႈ သို႔မဟုတ္ အခ်ိန္ ၾကန္႔ၾကာမႈ သို႔မဟုတ္ အမွား သို႔မဟုတ္ မတိက်မႈမ်ား ျဖစ္ပြားရာ အေျခအေနမ်ား သို႔မဟုတ္ ဝန္းက်င္မ်ားတြင္ အသံုးျပဳရန္ ရည္ရြယ္မထား သို႔မဟုတ္ သင့္ေလ်ာ္မႈမရွိျခင္း၊ HUAWEI ေဆာ့ဖ္ဝဲ သို႔မဟုတ္ ဝန္ေဆာင္မႈမ်ားမွ ပံ့ပိုးထားသည့္ ပါဝင္ခ်က္၊ ေဒတာ သို႔မဟုတ္ အခ်က္အလက္မ်ားသည္ အႏုျမဴ ပစၥည္းမ်ား လုပ္ေဆာင္ျခင္း၊ ေလယာဥ္ လမ္းညႊန္မႈ သို႔မဟုတ္ ဆက္သြယ္ေရး စနစ္မ်ား၊ ေလေၾကာင္း ခရီး ထိန္းခ်ဳပ္ျခင္း၊ အသက္ ပံ့ပိုးမႈ သို႔မဟုတ္ လက္နက္စနစ္မ်ားတို႔အပါအဝင္ ကန္႔သတ္မထားသည့္ ေသဆံုးမႈ၊ ကိုယ္တိုင္ ဒဏ္ရာရမႈ သို႔မဟုတ္ ႐ုပ္ပိုင္းဆိုင္ရာ သို႔မဟုတ္ သဘာဝပါတ္ဝန္းက်င္ဆိုင္ရာ အျပင္းအထန္ ထိခိုက္မႈတို႔ကို ျဖစ္ပြားေစႏိုင္သည္ဟု သင့္အေနျဖင့္ ေနာက္ထပ္ အသိအမွတ္ျပဳသေဘာတူပါသည္။
